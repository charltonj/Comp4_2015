CSS

Selectors Explained

CSS allows you to control both the style and the position of the content on your page.

By changing the CSS in one location all the elements using that styling can be changed at once.

The CSS selector determines which elements take on that styling.

Not discussed here are pseudo-selectors which can select items based on their state, or position in a document. They could be used for example to select the first line of a paragraph, the colour of a link when the mouse is over it, or the third column in a table. See this article at w3schools for more information.

Elemental Selectors

There are three simple selectors which operate on a named item. You can select based on an HTML tag, a user-defined class, or a user-defined id. While a class can be used many times in a document, an id should be used only once.

Example Notes
HTML tags
h1 { color: blue; } This will change all the <h1> headings to blue
body { color: white; } This changes all the text of the document to white!
b { color: red; } This will change everything in a <b> tag to red
Classes
.sarcastic { color: green; } This will change all elements of class 'sarcastic' to green
.warning { color: orange; } This will change all elements of class 'warning' to orange.
ID
#footer { background-color: lightgrey; } This changes the background of the element with id 'footer'.
#sidebar { border: 3px dotted green; } This will put a border around the div marked with id 'sidebar

Contained Selectors

These will select items inside another item. The first element is searched for an element of the second type. Note the space between the two tokens

Example Notes
table .comment { color: green; } This selects all the items of class 'comment' inside a <table>. Note the space!
#read_me_first em { color: white; } This selects all items tagged with <em> inside the element with id="read_me_first" (usually a <div>)

Elements of a given type

These will select items of a given class, usually. Here there is no space between the element and the class name.

Example Notes
table.results { color: darkslategray; } This will select any table that is of class 'results'.
p.comment { color: green; } This selects all paragraphs with a class of 'comment'.

Applying styles to multiple objects

You can apply the same style to multiple selections at once by seperating them with commas. A clean style is to put each selector on its own line, but remember not to put an extra comma after the last one!

Example Notes
h2, h3 em, p.revised { color: black; } This changes the colour of <h2> elements, emphasized elements inside an <h3>, and <p> tags of class 'revised'.
